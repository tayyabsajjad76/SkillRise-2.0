⚙️ Step 1: Start MongoDB Server
Open PowerShell / CMD and run: "C:\Program Files\MongoDB\Server\8.2\bin\mongod.exe" --dbpath="C:\data\db"

⚙️ Step 2: Run Backend Server
Open a new terminal (do NOT close MongoDB terminal) and run: npm run dev

To See DataBase in MongoDB:- http://localhost:3001/api/users
